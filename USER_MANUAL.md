# 🌍 AI Stock Predictor (Pro) - ユーザーマニュアル

## はじめに
このアプリケーションは、最先端のAI（ディープラーニング、機械学習）と伝統的なテクニカル分析、そしてファンダメンタルズ分析を融合させた、プロ仕様の株式予測・分析ツールです。
日本株、米国株、暗号資産に対応し、日々の有望銘柄の発掘からポートフォリオ構築、仮想売買による練習までをワンストップで提供します。

---

## 🚀 クイックスタート

### 起動方法
ターミナルで以下のコマンドを実行してください。
```bash
streamlit run app.py
```
ブラウザが自動的に開き、アプリが表示されます（通常は `http://localhost:8501`）。

## 📱 機能ガイド

### 1. 📊 市場スキャン (Market Scan)
市場全体をスキャンし、AIとテクニカル指標に基づいて「今、買い時/売り時」の銘柄を抽出します。

*   **今日のイチオシ**: 最も期待値が高い銘柄をピックアップ表示します。
*   **シグナル一覧**: 各戦略（SMA, RSI, Bollinger, AI, LSTMなど）に基づく売買シグナルをカード形式で表示します。
*   **チャートパターン検出**: ダブルボトムや逆三尊などの形状を自動検出し、チャート上に可視化します。
*   **ファンダメンタルズ・フィルター**: サイドバーでPER/PBR/ROEの条件を設定し、業績の悪い銘柄を除外できます。

### 2. 💼 ポートフォリオ (Portfolio)
現代ポートフォリオ理論 (MPT) に基づき、最適な資産配分を提案します。

*   **効率的フロンティア**: リスク・リターン平面上で、最も効率的なポートフォリオ（シャープレシオ最大など）を可視化します。
*   **相関行列**: 銘柄間の連動性をヒートマップで確認し、分散効果を高める組み合わせを探せます。
*   **ロボアドバイザー**: AIが推奨するポートフォリオ配分を自動計算します。

### 3. 📝 ペーパートレード (Paper Trading)
仮想資金（初期1,000万円）を使って、リスクなしでトレードの練習ができます。

*   **注文**: 市場スキャン結果からワンクリックで注文、または手動で銘柄コードを指定して注文できます。
*   **ポジション管理**: 保有銘柄の評価損益をリアルタイムで確認できます。
*   **取引履歴**: 過去の売買記録を一覧表示します。

### 4. 📈 ダッシュボード (Dashboard)
あなたのトレード成績を詳細に分析します。

*   **資産推移**: 日経平均（ベンチマーク）と比較した資産の増減グラフ。
*   **戦略別パフォーマンス**: どの戦略が最も利益を上げているかを分析。
*   **月次リターン**: 月ごとの勝敗を可視化。

### 5. 🕰️ 過去検証 (Historical Backtesting)
過去10年分のデータを使って、戦略の長期的な有効性を検証します。

*   **Buy & Hold比較**: ガチホした場合とAIトレードした場合のパフォーマンスを比較できます。

## ⚙️ 設定 (Sidebar)

*   **市場選択**: 日本株 (Japan)、米国株 (US)、暗号資産 (Crypto) などを切り替えられます。
*   **分析期間**: 過去データ取得期間（1y, 2y, 5y）を選択します。
*   **取引設定**: 単元未満株（1株単位）での取引を有効にできます。
*   **通知設定**: LINE Notify や Discord Webhook を設定すると、毎日のスキャン結果がスマホに届きます。
*   **自動更新 (Live Mode)**: ONにすると、60秒ごとに画面を自動更新し、最新価格を反映します。

---

## ⚠️ トラブルシューティング

*   **"Ticker not found" エラー**: 銘柄コードが正しいか確認してください（日本株は末尾に `.T` が必要です）。
*   **スキャンが遅い**: ディープラーニング戦略や「5y」データを選択している場合、計算に時間がかかります。分析期間を短くするか、対象銘柄を絞ってください。
*   **通知が届かない**: トークンが正しいか、インターネット接続があるか確認してください。
