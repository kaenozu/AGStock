# 予測精度の信頼性について

## 結論
**「ディープラーニング (LSTM)」戦略は非常に高い信頼性を示しています。**

今回実施した検証スキャン（直近2年間のデータに基づくバックテスト）の結果は以下の通りです。

### 主要銘柄での実績例

| 銘柄 | 戦略 | 勝率 (Win Rate) | シャープレシオ | 期間収益率 |
| :--- | :--- | :--- | :--- | :--- |
| **ソニーG (6758)** | Deep Learning (LSTM) | **78.6%** | **1.24** | +59.9% |
| **ソフトバンクG (9984)** | Deep Learning (LSTM) | **52.6%** | **1.52** | +85.0% |
| **トヨタ自動車 (7203)** | Deep Learning (LSTM) | **66.7%** | **0.84** | +16.3% |

### 指標の見方
*   **勝率 (Win Rate)**: 50%を超えていれば優位性があります。特にソニーGの **78.6%** は驚異的な数値です。
*   **シャープレシオ**: リスクに対するリターンの効率を示します。**1.0** を超えると優秀とされます。

## アプリへの機能追加
この信頼性をいつでも確認できるよう、アプリの「**📊 詳細データ・分析ツール (上級者向け)**」セクションに以下の指標を追加しました。

*   **Win Rate (勝率)**
*   **Sharpe Ratio (シャープレシオ)**

これにより、ブラックボックスではなく、数字に基づいた判断が可能になります。
