2. 初期資金は? → 金額を入力
3. リスク許容度は? → 低/中/高

## 💡 よくある質問 (FAQ)

### Q1: 本当に利益が出ますか?

**A:** 
- ペーパートレード (仮想取引) なので実際のお金は動きません
- まずは1ヶ月試して、利益が出るか確認してください
- 利益が安定したら、実際の取引を検討してください

### Q2: 朝5時に起きられません

**A:**
- 統合ダッシュボードはいつでも開けます
- 朝7時でも、夜でもOK
- 自分の生活リズムに合わせてください

### Q3: 通知が来ません

**A:**
1. `config.json` を確認
2. `notifications.enabled` が `true` か確認
3. LINEトークンが正しいか確認
4. テスト送信してみる

### Q4: エラーが出ます

**A:**
1. エラーメッセージをコピー
2. `logs/auto_trader.log` を確認
3. よくあるエラー:
   - データ取得失敗 → インターネット接続確認
   - ポート使用中 → 別のポートで起動

### Q5: 設定を変更したい

**A:**
```bash
python setup_wizard.py
```
もう一度実行すれば、設定を上書きできます

### Q6: バックアップは?

**A:**
重要なファイル:
- `config.json` - 設定
- `paper_trading.db` - 取引履歴
- `stock_data.db` - 株価データ

定期的にコピーして保存してください
