## 🎯 改善のポイント

### 1. 情報の優先順位
```
最重要 → 総資産と今日の変動
重要   → 保有銘柄の状況
普通   → グラフや履歴
```

### 2. 色の使い方
```
🟢 緑 = 利益、良い状態
🔴 赤 = 損失、注意
⚪ 白 = 中立
```

### 3. 言葉の簡素化
```
❌ 「unrealized_pnl_pct」
✅ 「+2.5%」

❌ 「ポジション」
✅ 「保有銘柄」

❌ 「エクイティカーブ」
✅ 「資産の推移」
```
