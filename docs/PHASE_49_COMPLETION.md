## 🎯 主要機能

### 5ステップの質問

#### ステップ1: 投資経験 👤
- 初心者 (1年未満)
- 中級者 (1-3年)
- 上級者 (3年以上)

#### ステップ2: 初期資金 💰
- 経験レベルに応じたデフォルト値
- 最低10万円から設定可能
- カンマ区切り対応

#### ステップ3: リスク許容度 🎯
- 低: 安全第一 (年間目標3-5%)
- 中: バランス重視 (年間目標5-10%)
- 高: 積極的 (年間目標10%以上)

#### ステップ4: 通知方法 📱
- LINE (推奨)
- Discord
- メール
- なし

**LINE設定:**
- トークン取得方法を表示
- その場で設定可能
- 後から設定も可能

#### ステップ5: 自動化レベル 🤖
- 手動: すべて自分で判断
- 半自動: AIの推奨を確認して承認 (推奨)
- 全自動: AIに完全おまかせ (上級者向け)

**安全機能:**
- 全自動選択時に再確認
- 初心者には半自動を推奨

## ⚙️ 自動生成される設定

### リスクレベル別パラメータ

#### 低リスク (安全第一)
```json
{
  "stop_loss_pct": 0.03,        // 損切り 3%
  "take_profit_pct": 0.05,      // 利確 5%
  "max_position_size": 0.10,    // 最大ポジション 10%
  "daily_loss_limit_pct": -3.0  // 日次損失制限 -3%
}
```

#### 中リスク (バランス)
```json
{
  "stop_loss_pct": 0.05,        // 損切り 5%
  "take_profit_pct": 0.10,      // 利確 10%
  "max_position_size": 0.15,    // 最大ポジション 15%
  "daily_loss_limit_pct": -5.0  // 日次損失制限 -5%
}
```

#### 高リスク (積極的)
```json
{
  "stop_loss_pct": 0.07,        // 損切り 7%
  "take_profit_pct": 0.15,      // 利確 15%
  "max_position_size": 0.20,    // 最大ポジション 20%
  "daily_loss_limit_pct": -7.0  // 日次損失制限 -7%
}
```

### 経験レベル別の対象資産

- **初心者**: 日本株のみ
- **中級者**: 日本株 + 米国株
- **上級者**: 日本株 + 米国株 + 欧州株
