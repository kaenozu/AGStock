# 🚀 AGStock - 3ステップで始める

## ステップ1: 起動する（10秒）

**Windowsの場合:**
1. `run_unified_dashboard.bat` をダブルクリック
2. ブラウザが自動的に開きます

**または、手動で:**
```bash
streamlit run unified_dashboard.py
```

ブラウザで `http://localhost:8501` が開きます。

## ステップ3: 何もしない（重要！）

### 最初の3日間は「見るだけ」でOK

**やること:**
- ✅ 毎朝、ダッシュボードを開く
- ✅ 「朝活」タブを見る
- ✅ AIの推奨を読む

**やらないこと:**
- ❌ 「承認」ボタンを押さない
- ❌ 設定を変更しない
- ❌ 焦らない

**理由:** システムの動きを理解するため

---

## 4日目以降: 少しずつ試す

## よくある質問

### Q1: 本当にお金が動く？
**A:** いいえ！これは「ペーパートレード（練習）」です。
仮想のお金（100万円）で練習できます。

### Q2: 何も表示されない
**A:** 以下を確認:
1. `run_unified_dashboard.bat` を実行したか？
2. ブラウザが開いたか？
3. `http://localhost:8501` にアクセスしているか？

### Q3: エラーが出る
**A:** 
```bash
# これを実行してみてください
python production_readiness_check.py
```

## 困ったら

### 再起動する
1. ブラウザを閉じる
2. `run_unified_dashboard.bat` をもう一度ダブルクリック

### 初期化する
```bash
python setup_wizard.py
```
質問に答えるだけで設定完了！
